Gemini TTS CLI

A CLI for converting text to speech using the Google Gemini TTS API.

Setup

  1. Install dependencies: 
    npm install @google/genai mime chalk dotenv
npm install -D @types/node tsx
  2. Create a .env file: 
    GEMINI_API_KEY=your_google_genai_api_key_here
  3. Run the CLI: 
    npm run tts

Usage

  • Enter text to synthesize
  • Choose speaker: 1 (Zephyr), 2 (Puck)
  • Optional style (e.g. Warm, Calm)
  • Audio saved as output_audio.wav
  • Type exit to quit

Scripts

  • test_api.ts — basic test
  • interactive_tts.ts — simple CLI
  • tts_advanced_cli.ts — full CLI
